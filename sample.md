# Markdown 预览器 Demo

这是一个用 **Tauri** 写的、只用来看 Markdown 的小应用。

## 功能

- 点按钮打开 `.md` 文件
- 直接把文件拖进窗口
- 支持 GFM、代码高亮

## 代码示例

```rust
fn main() {
    println!("Hello, Tauri!");
}
```
